# vscode-devenv

## getting started

- install podman desktop (on Windows) or orbstack (on macOS)
- install vscode
- in vscode:
  - open command center and run `remote-containers.openRepositoryInUniqueVolume`
  - choose github
  - select `boozedog/vscode-devenv`
- ... profit!

## building

```sh
devenv test
```
